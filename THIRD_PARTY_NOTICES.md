# Third Party Notices

Este arquivo registra todo software open-source de terceiros absorvido pelo Hermes 2.0,
seja como dependência (vendor), reescrita inspirada (port), processo externo (wrap),
ou documentação de uso (skill-only).

Nenhuma absorção ocorre sem passar pelo License Gate (A3) da skill `oss-absorb`.

---

## Estrutura

```
third_party/
├── <nome-do-projeto>/
│   ├── LICENSE          (cópia da licença original, se exigido)
│   └── NOTICE           (aviso de copyright, se aplicável)
└── README.md            (este arquivo)
```
