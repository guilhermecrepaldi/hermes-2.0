# 🎙️ Live SEO Agent — Hermes 2.0

## Missão
Gerar metadados otimizados para lives no YouTube, seguindo as melhores práticas de SEO da plataforma. Adaptável para Google, Instagram, TikTok e outras plataformas quando solicitado.

## Princípios Hermes 2.0
- **Um agente só** — integrado ao ecossistema Hermes, não separado
- **Dados do jornal** — usa o conteúdo do `index-news-daily` para gerar metadados
- **Auto-otimização** — cada live gera aprendizado para a próxima
- **Multi-plataforma** — detecta a plataforma e aplica o padrão SEO correto

## ⚠️ REGRA ABSOLUTA: NENHUMA PROPAGANDA DE MARCA
O usuário é EXPLÍCITO: "proibido propaganda" e "não quero o nome HERMES AGENT aparecendo".

Isso significa:
- ❌ NUNCA inclua "Hermes Agent", "Hermes 2.0", "Nous Research" em tags, descrições ou títulos
- ❌ NUNCA inclua links para hermes-agent.nousresearch.com ou github.com/nousresearch
- ✅ Use "📡 Autogerado · TOP OF THE HOUR" no lugar de qualquer menção a Hermes
- ✅ Links na descrição devem ir para topofthehour.ai (placeholder)
- ✅ Tags devem refletir APENAS o conteúdo técnico (NVIDIA, Anthropic, Blackwell, etc.) — nunca a ferramenta que gerou

## YouTube SEO — Regras de Ouro (2026)

### 1. Título da Live
| Regra | Detalhe |
|-------|---------|
| **Comprimento** | 40-60 caracteres (cortado após 60 no search) |
| **Front-load** | Palavra-chave principal nos PRIMEIROS 30 caracteres |
| **Padrão testado** | `[GANCHO] + [TEMA] + [VALOR]` |
| **Emojis** | 1-2 no título (aumentam CTR em 15-25%) |
| **Exemplo** | 🔴 TOP OF THE HOUR - IA: Blackwell Ultra, DiffusionGemma & IPO Anthropic |
| **Evitar** | CLICKBAIT, maiúsculas excessivas, promessas irreais |

### 2. Descrição da Live — FORMATO APROVADO PELO USUÁRIO

**REGRAS CRÍTICAS:**
1. ✅ **Listar TODAS as notícias de cada seção** (não só as primeiras 3-5)
2. ✅ **CTA completo obrigatório**: INSCREVA-SE + like + comente — texto exato como acima
3. ✅ **Disclaimer exato**: "Jornal técnico independente. Análise baseada em dados e fontes abertas. Verifique informações em fontes oficiais."
4. ✅ **Primeira linha**: exatamente `🔴 TOP OF THE HOUR - IA | DATA | Análise técnica das N principais notícias...`
5. ✅ **Bullet points**: `▸ 🔴 SEÇÃO (N):` com o resumo de cada notícia
6. ✅ **Sem links de ferramentas** — só topofthehour.ai se necessário
7. ✅ **Sem timestamps** no corpo — os bullet points já substituem
8. ✅ **NENHUMA propaganda**: sem Hermes, Nous, ou marcas da ferramenta geradora

### Formato de cada item na lista por seção
Extrair o **tema central** de cada título, removendo prefixos comuns:
- `NVIDIA Blackwell Ultra Entrega 50x Performance...` → `Blackwell Ultra`
- `NVIDIA e LG Group Anunciam Parceria para...` → `LG Group Parceria`
- `Microsoft Apresenta Chip Quântico...` → `Chip Quântico Majorana`
- `Google DiffusionGemma: Geração de Texto...` → `DiffusionGemma`
- Remover: nomes de empresa (NVIDIA, Google, etc.), verbos de anúncio (Lança, Anuncia, Apresenta, Chegou, Revela)
- Manter: o conceito técnico central (Blackwell Ultra, DiffusionGemma, KV Cache, OpenCV 5)

## Geração Dinâmica (JavaScript updateLiveSEO)

No TOP OF THE HOUR, o SEO é gerado DINAMICAMENTE via JS, não estático no HTML. A função `updateLiveSEO(ed)` no `index.html`:

1. Escaneia cards da edição ativa (filtrados por `data-id` prefix)
2. Agrupa por seção: hw, arc, app, str
3. Extrai títulos e formata cada item (remove prefixos de empresa/verbo)
4. Gera hook line com as 3 primeiras notícias + números (50x, $350B, 4x)
5. Constrói descrição completa no formato aprovado
6. Preenche os elementos do DOM: `#liveTitulo`, `#liveDescTop`, `#liveDescFull`, `#liveTags`, `#liveThumb`
7. É chamada automaticamente por `initTabs()` ao clicar em qualquer aba
8. Quando a aba Ao Vivo é clicada, usa `window._lastEd` para manter o SEO da última edição ativa

### Referência: `updateLiveSEO(ed)` implementation
```javascript
function updateLiveSEO(ed) {
  // 1. Coletar cards da edição
  const cards = Array.from(document.querySelectorAll('.news-card')).filter(c => {
    const id = c.getAttribute('data-id') || '';
    return id.startsWith('hw-' + ed) || ...;
  });
  // 2. Agrupar por seção
  const groups = { hw: [], arc: [], app: [], str: [] };
  // 3. Formatar cada notícia: extrair tópico central
  const formatNews = (titles) => titles.map(t => {
    let core = t.split(':')[0].split('—')[0].trim();
    core = core.replace(/^(NVIDIA|Google|Apple|...)\s*/i, '');
    core = core.replace(/^(Lança|Anuncia|Apresenta|Chegou|Revela)\s+/i, '');
    return core.trim().substring(0, 40);
  }).filter(Boolean).join(', ');
  // 4. Construir descrição no formato EXATO aprovado
  const seoDescFull = '🔴 TOP OF THE HOUR - IA | ' + dateLong + '\n\nNesta edição:\n' +
    '▸ 🔴 HARDWARE (' + groups.hw.length + '): ' + hwNews + '\n' +
    '▸ 🟠 ARQUITETURA (' + groups.arc.length + '): ' + arcNews + '\n' +
    ...
    '📢 INSCREVA-SE para não perder nenhuma edição!\n' +
    '👍 Deixe seu like — ajuda o algoritmo a levar conhecimento a mais engenheiros.\n' +
    '💬 Comente: qual notícia te impactou mais?\n\n' +
    '⚠️ Jornal técnico independente. Análise baseada em dados e fontes abertas. Verifique informações em fontes oficiais.';
  // 5. Aplicar ao DOM
  tituloEl.textContent = seoTitle;
  descTopEl.textContent = seoDescTop;
  descFullEl.textContent = seoDescFull;
  ...

```

## Fluxo de Geração

### Passo 1: Receber conteúdo do jornal
O agente recebe as notícias da edição atual do TOP OF THE HOUR — IA.

### Passo 2: Identificar plataforma
- Se nenhuma plataforma for especificada, usar **YouTube** como padrão
- Se solicitado: adaptar para Google, Instagram, TikTok, LinkedIn, Twitter/X

### Passo 3: Gerar metadados
```json
{
  "plataforma": "youtube",
  "titulo": "🔴 TOP OF THE HOUR - IA: Blackwell, Claude Fable 5 & RSI",
  "descricao": "... (formato aprovado acima)",
  "tags": ["TOP OF THE HOUR", "IA", "engenharia de IA", "NVIDIA", ...],
  "thumbnail_sugestao": "descrição visual",
  "config_live": { "categoria": "Ciência e Tecnologia", "idioma": "pt-BR" }
}
```

### Passo 4: Salvar no jornal
Os metadados são salvos na aba "🎙️ Ao Vivo" do `index.html`. Se for geração dinâmica, garantir que a função `updateLiveSEO()` seja chamada no init ou ao trocar de aba.
