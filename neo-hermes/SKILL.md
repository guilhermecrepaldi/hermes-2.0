## Como Usar

### Iniciar Sessão Neo Hermes

```bash
# Carregar tudo de uma vez
skill_view(name='auto-assembly-workflow')
skill_view(name='conselho-ias')
skill_view(name='dag-workflow')
skill_view(name='token-budget-control')
skill_view(name='auto-executor')
skill_view(name='auto-healing')
skill_view(name='roteador-economico')
skill_view(name='output-coeso')

# Pronto! Agora é só descrever tarefas.
```

### Ou usar o Auto-Assembly (recomendado)

```bash
skill_view(name='auto-assembly-workflow')
# Ele carrega as skills necessárias automaticamente
```

### Exemplos de Uso

```
👤 "Cria uma API de usuários com FastAPI, testa e faz deploy"
   → Auto-Assembly detecta: projeto novo
   → Monta: Conselho (Arquiteto+Revisor) → DAG (backend||tests) → Deploy
   → Budget: Normal ($0.002)
   → Resultado: API no ar em 2 minutos

👤 "Refatora o módulo de pagamentos e adiciona testes"
   → Auto-Assembly detecta: refatoração
   → Monta: Conselho (4 papéis) → DAG (refatorar||testar) → Auditor
   → Budget: Reflexão ($0.004)
   → Resultado: Código refatorado com 90%+ cobertura

👤 "Pesquisa as trends de IA dessa semana"
   → Auto-Assembly detecta: pesquisa
   → Monta: Conselho (Arquiteto+Revisor) → Web parallel
   → Budget: Profundo ($0.007)
   → Resultado: Relatório consolidado
```
